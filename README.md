# Kosovo Election Calculator

A web-based calculator for Kosovo parliamentary elections using the D'Hondt method with 100 regular seats + 20 reserved seats for minorities.

## Features

- Calculate seat distribution using the D'Hondt method
- 5% election threshold
- Real-time results display
- Responsive design with Tailwind CSS
- No server-side requirements (runs entirely in browser)

## How to Use

1. Enter vote percentages for each party (LVV, LDK, PDK, AAK, NISMA)
2. Click "Llogarit Mandatet" (Calculate Seats)
3. View the seat distribution results

## Algorithm

- Uses D'Hondt method for 100 seats
- 5% threshold for party eligibility
- 20 reserved seats for minorities automatically added
- Calculated entirely client-side with JavaScript
